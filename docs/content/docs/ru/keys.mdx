## Режимы трекбола

Цвет подсветки всегда показывает активный режим.

| Режим | Шарик… | Вкл/выкл | Подсветка |
|-------|--------|----------|-----------|
| **Курсор** | двигает указатель мыши, ускоряясь со скоростью | всегда включён | обычная |
| **Снайп** | двигает указатель медленно и точно, без ускорения | держать `SNIPING` · переключить `SNP_TOG` · сам, пока зажат `PT_Z`/`PT_SLSH` | 🔴 красный |
| **Drag-scroll** | скроллит, держась за одну ось; флик докручивает по инерции | держать `DRGSCRL` · переключить `DRG_TOG` · тап `DRGT_CRTT` | 🟢 зелёный |
| **Каретка** | двигает текстовый курсор стрелками; при снайпе прыгает по словам | держать `CARRETM` · переключить `CRT_TOG` · холд `DRGT_CRTT` | 🔵 лазурный |

Drag-scroll и каретка взаимоисключающие — включение одного выключает другой.

## Скраб-клавиши (держать + крутить шарик)

Живут на LOWER-слое (левый большой палец), на левой руке — правая остаётся
на шарике.

| Клавиша | Пока держится, шарик… | Где (LOWER) |
|---------|------------------------|-------------|
| `SCRB_TAB` | переключает вкладки (Ctrl+PgUp/PgDn) | средний ряд, внутренняя колонка |
| `SCRB_VOL` | меняет громкость | верхний ряд, внутренняя колонка |
| `SCRB_SPC` | переключает рабочие столы macOS (Ctrl+←/→) | нижний ряд, внешняя колонка |
| `SCRB_HST` | листает undo/redo (⌘Z / ⇧⌘Z) | нижний ряд, 2-я колонка |
| `SCRB_APP` | ходит по переключателю приложений (Cmd/Alt+Tab по ОС, мод держится сам) | нижний ряд, 3-я колонка |

## Pointer-слой

Держится через `PT_Z` (Z) или `PT_SLSH` (/). Ядро 3x5, одинаковое на всех
платах:

```text
QK_BOOT  DPI_RMOD DPI_MOD  S_D_RMOD S_D_MOD  │  S_D_MOD  S_D_RMOD DPI_MOD  DPI_RMOD QK_BOOT
LGUI     LALT     LCTL     LSFT     DRGT_CRTT│  CRT_TOG  RSFT     RCTL     RALT     RGUI
 · ·     DRGSCRL  SNIPING  DRG_TOG  SNP_TOG  │  SNP_TOG  DRG_TOG  SNIPING  DRGSCRL   · ·
                  BTN2     BTN1     BTN3     │  BTN_LOCK BTN1
```

`DPI_MOD`/`DPI_RMOD` — DPI курсора выше/ниже, `S_D_MOD`/`S_D_RMOD` — то же
для DPI снайпа.

## Кастомные кейкоды

Объявлены в `users/wh1teend/common.h`, обрабатываются в `common.c`.
Веб-конфигуратор показывает их как неизвестные коды; назначаются через
кнопку **Any** по hex-значению.

<Callout type="warn" title="Правки из браузера не переживают прошивку">
Слои пересеиваются из `keymap.c` при каждой новой сборке. Комбо, тапдансы
Argos и настройки DPI лежат в других областях EEPROM и выживают.
</Callout>

| Клавиша | Hex (Any) | Поведение |
|---------|-----------|-----------|
| `DRGT_CRTT` | `0x7E40` | Тап — вкл/выкл drag-scroll · Холд — вкл/выкл каретку |
| `CARRETM`   | `0x7E41` | Каретка, пока удерживается (в сетках не стоит — запасной) |
| `CRT_TOG`   | `0x7E42` | Вкл/выкл каретку |
| `CRT_SEL`   | `0x7E43` | Пока удерживается: каретка + Shift — шарик тянет выделение текста (в сетках не стоит — запасной) |
| `BTN_LOCK`  | `0x7E44` | Защёлка кнопки мыши 1: тащишь, не удерживая клавишу |
| `SCRB_TAB`  | `0x7E45` | Скраб: вкладки |
| `SCRB_VOL`  | `0x7E46` | Скраб: громкость |
| `SCRB_SPC`  | `0x7E47` | Скраб: рабочие столы macOS |
| `SCRB_HST`  | `0x7E48` | Скраб: undo/redo |
| `SCRB_APP`  | `0x7E49` | Скраб: переключатель приложений (мод держится сам) |

Печатные бонусы: двойной тап Shift включает **Caps Word** (капс до конца
слова; `CW_TOGG` стоит на RAISE рядом с `EE_CLR`), а **`QK_REP`** правее —
повтор последней клавиши.

## RGB-индикация

| Цвет | Состояние |
|------|-----------|
| 🔵 Лазурный | Каретка |
| 🟢 Зелёный | Drag-scroll |
| 🔴 Красный | Снайп |
| 🟠 Оранжевый | Защёлкнута кнопка мыши 1 (`BTN_LOCK`) |
| ⚪ Белый | Активен Caps Word |
| 🟡 Золотой | Зажат LOWER-слой |
| 🟣 Фиолетовый | Зажат RAISE-слой |
| Обычная | Ничего не включено |

Список — это и порядок приоритета: режимы трекбола важнее защёлки, защёлка —
Caps Word, Caps Word — слоёв. Подсветка гаснет после 5 минут простоя
(`RGB_MATRIX_TIMEOUT`).
